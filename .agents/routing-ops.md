# Routing Operations

## Channel-Based Routing
Messages are routed based on which Teams channel they appear in:
- **Fabric Docs** channel → fabric-docs agent → `F:\git\fabric-docs-pr`
- **Foundry Docs** channel → foundry-docs agent → `F:\git\azure-ai-docs-pr`
- **Home** channel → home agent → `F:\home`

## Response Rule
ALL responses MUST be prefixed with "Copilot: " to distinguish AI responses from Jon's messages.
